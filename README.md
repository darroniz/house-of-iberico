# House of Ibérico — by Omar Allihboy

Propuesta de diseño para la sección *House of Ibérico / The Ibérico School* de [ibericosense.co.uk](https://ibericosense.co.uk/).

HTML/CSS/JS estático, sin dependencias. Identidad visual del sitio: rojo `#AC1735`, Acumin Variable Concept y Lovelace Script (fuentes del propio sitio, autoalojadas en `fonts/`).

## Estructura

1. **Hero** — panel rojo + foto de Omar en la Dehesa.
2. **01 The Ambassador** — bio, cifras de campaña y los 4 films (Nature, Flavour, Sustainability, Time).
3. **02 The Recipes** — pestañas por categoría con lightbox de vídeo (Bunny Stream). Mediterranean con fotos y vídeos reales; el resto, tarjetas a la espera de assets.
4. **03 The Art of Carving** — índice de 7 píldoras + master class (coming soon).
5. **04 The Production Process** — 6 capítulos + master class (coming soon).
6. **CTA distribuidores** + footer.

## Ver en local

```
python3 -m http.server 8000
```

y abrir `http://localhost:8000`.
